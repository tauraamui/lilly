# project petal (bobatea re-write)

This branch is a from scratch clean start for the Lilly project. There is one main motivation for this re-write, and it is to utilise the new [bobatea](https://github.com/tauraamui/bobatea) TUI library (basically bubblatea for V).
Having worked at charm_ for a little while now, I have already learnt so much about how to design well constructed and nicer looking terminal applications. The bobatea library is intended
to be a bubbletea equivilant for those wanting to utilise V instead of Go. This editor is intended to be *the* reference implementation of programs written with bobatea.
